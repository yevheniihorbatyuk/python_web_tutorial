# Python Web Course Materials

Навчальний репозиторій для занять з Python web/data. Кожен модуль самодостатній: містить код, Docker/infra, конспекти та сценарій уроку. Використовуйте як матеріал для занять або швидкий старт для власних pet/production-проєктів.

## Модулі
- `module_06` — реляційні БД, PostgreSQL, asyncio та production ETL патерни.
- `module_08` — SQLAlchemy ORM + NoSQL (MongoDB), кеші, черги RabbitMQ, асинхронні flow для DS/DE.

## Як користуватися
1. Оберіть папку модуля і прочитайте його `README.md` або `docs/START_HERE.md`.
2. Скопіюйте `.env.example` → `.env`, запустіть `docker-compose up -d` (якщо потрібно).
3. Виконайте кроки з `docs/QUICKSTART.md`, потім переходьте до практик з `README.md`.
4. Для поглиблення — відкрийте `docs/ADVANCED_README.md` та `docs/LESSON_PLAN.md`.

## Що всередині
- Короткий конспект і cheat sheet по темі.
- Прикладний код з фокусом на реальні кейси DS/DE: ETL, події, кешування, інтеграції з чергами і документними БД.
- Окремі приклади для синхронної та асинхронної роботи, щоб показати різницю в архітектурних рішеннях.

## Подальша модифікація
- Додавайте нові модулі в корінь репозиторію як `module_XX` із власними `docs/` та прикладами.
- Зберігайте однаковий стиль документації: `START_HERE`, `QUICKSTART`, `LESSON_PLAN`, `SUMMARY`, `ADVANCED_README`, `PROJECT_OVERVIEW`.
- Для нових сервісів додавайте сервіси в `docker-compose.yml` конкретного модуля, а секрети — у `.env`.

Кодові приклади написані з пріоритетом читаності та можливістю швидко «витягнути» окремі фрагменти у ваші робочі проєкти.
