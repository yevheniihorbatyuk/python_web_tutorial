# Урок 1: Основи веб-скрапінгу з Beautiful Soup

## Вступ: Навіщо потрібен веб-скрапінг?

Веб-скрапінг — це автоматизований процес видобування даних з веб-сайтів. Це потужна техніка для збору інформації для моніторингу цін, агрегації новин, дослідження ринку та багато іншого.

**Проблема:** Дані в Інтернеті вбудовані в складний HTML-код. Ручне копіювання цих даних є повільним і неефективним.

**Рішення:** Ми використовуємо бібліотеки, такі як **Beautiful Soup**, щоб парсити HTML і програмно видобувати саме ті дані, які нам потрібні.

---

## Частина 1: Основи HTML та CSS-селекторів

Щоб скрапити веб-сайт, потрібно розуміти його структуру.

### HTML як дерево

Кожен HTML-документ — це вкладене дерево тегів.

```html
<div class="quote">
    <span class="text">"Світ, яким ми його створили..."</span>
    by <small class="author">Альберт Ейнштейн</small>
</div>
```

- **Теги:** `<div>`, `<span>`, `<small>`
- **Атрибути:** `class="quote"`, `class="text"`
- **Вміст:** Текст усередині тегів.

### Пошук елементів за допомогою CSS-селекторів

CSS-селектори — це шаблони, що використовуються для вибору елементів.

- `div`: вибирає всі теги `<div>`.
- `.quote`: вибирає всі теги з `class="quote"`.
- `div.quote`: вибирає теги `<div>`, які мають `class="quote"`.
- `span.text`: вибирає теги `<span>` з `class="text"`.

## Частина 2: Процес скрапінгу

Процес включає дві основні бібліотеки:

1.  **Requests**: для завантаження HTML-вмісту веб-сторінки.
2.  **Beautiful Soup**: для парсингу HTML та пошуку даних.

### Покроковий робочий процес

1.  **Встановіть бібліотеки**: `pip install requests beautifulsoup4`
2.  **Завантажте сторінку**: Використовуйте `requests.get(url)` для отримання HTML.
3.  **Створіть об'єкт Soup**: `soup = BeautifulSoup(html_content, 'html.parser')`
4.  **Знайдіть елементи**: Використовуйте `soup.find()` або `soup.find_all()` з CSS-селекторами.
5.  **Видобудьте дані**: Отримайте текст за допомогою `.get_text()` або значення атрибутів за допомогою `['attribute_name']`.

---

## Частина 3: Ключові методи Beautiful Soup

- **`soup.find('tag', class_='name')`**: знаходить *перший* відповідний елемент.
- **`soup.find_all('tag', class_='name')`**: знаходить *усі* відповідні елементи та повертає їх у вигляді списку.
- **`element.get_text()`**: видобуває текстовий вміст з елемента.
- **`element.get('href')`**: видобуває значення атрибута (наприклад, URL з посилання).

## Частина 4: Етика скрапінгу

- **Перевіряйте `robots.txt`**: Завжди перевіряйте `website.com/robots.txt`, щоб побачити, які частини сайту дозволено скрапити.
- **Будьте ввічливими**: не надсилайте занадто багато запитів за короткий проміжок часу. Додавайте затримки (`time.sleep()`) між запитами.
- **Представляйтеся**: встановіть `User-Agent` у заголовках запиту, щоб ідентифікувати ваш скрипт.
